# SmartService Website

SmartService is a full-stack service booking website where users can:

- Register and Login
- Book Services
- Contact via Email
- View Dashboard
- Track Upcoming and Completed Bookings

---

# Features

## User Authentication
- Secure Registration
- Login System
- MySQL Database Authentication

## Booking System
- Book Services
- Booking Status System
- Upcoming & Completed Tracking

## Dashboard
- Total Bookings
- Upcoming Bookings
- Completed Bookings

## Contact System
- Email Integration using Nodemailer

---

# Technologies Used

## Frontend
- HTML
- CSS
- JavaScript

## Backend
- Node.js
- Express.js

## Database
- MySQL

---

# Installation

## Clone Repository

```bash
git clone (https://github.com/esha1076/smartservice.git)
```

## Install Dependencies

```bash
npm install
```

## Start Server

```bash
node server.js
```
